Project Overview

Our project implements different types of slide transitions in a few different functions. Given two input images of the same size, a start image and an end image, our project contains multiple functions that each generate a different transition between the two, including a sliding transition, a fade transition, a circle transition, a flash transition, and a split transition. The transition is generated as a set of images in a directory specified as a parameter in the method, which can be combined in a video editor to create a cohesive video.


Procedure

This section outlines the low level workflow procedure of our project. It will be delineated in steps. Here is the python script:

https://drive.google.com/file/d/1056LxlLksZx24cy4wyGRe9hALBPow79c/view?usp=sharing

Inputs: Two images of the same size, a start and and end image (we assume both input images are of the same size)

Create and navigate to a directory called yourDirectory (or whatever name you please) somewhere in your files where you would like your transition sequences to be saved. This directory will hold other directories containing transition sequences. Record the path name to this directory, ensuring that it ends in a slash.

        Example: /Users/StevenLi/Desktop/CS4475/Project1/outputs/

Add two images called start.ext and end.ext into yourDirectory. Ensure they have the same size. If they do not, crop them before continuing the following steps. Also, add the transition.py file to the directory.

Within yourDirectory, create another directory called yourTransition (or whatever name you please). Record the name of this directory, ensuring that it ends in a slash. This will be the directory in which the transition sequence is saved.

        Example: slideoutputsequence/

Open the transitions.py script. At the top of the file, input the path names of yourDirectory and yourTransition you recorded earlier into the indicated variables. At the bottom of the file, input the names of the image files into the indicated variables. At the very bottom, uncomment the function corresponding to the transition you would like to perform. Edit the frameCount parameter depending on how many total images you would like in your final sequence.

In your command line, run they command python transitions.py

Check yourTransition. It should contain new images.

Open a video editing application of your choice. Import all images in the sequence as static images in order. Decrease the time frame of each image to a fraction of a second (no longer than 0.1 seconds per image). Set the transition between images to none.

Export the video from the image editor. You will be left with a video of the transition.

Optionally, if the video editor displays each image for too long, you can speed the video up after exporting.

Outputs: A video of the desired transition from the start image to the end image
